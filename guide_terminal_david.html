🔖 Guide — Lancer l'app
À faire chaque fois que tu veux travailler sur Highlights
Étape 1 — Ouvrir le terminal
Option A — Le plus simple
Via l'explorateur de fichiers
Navigue jusqu'à ton dossier highlights-app, puis dans la barre d'adresse en haut tape cmd et appuie sur Enter. Le terminal s'ouvre directement dans le bon dossier.
Option B — Via le terminal Windows
Windows + R → cmd → Enter
Puis navigue au bon dossier avec cette commande :
cd "C:\Users\User\Desktop\Claude Cowork folder\highlights-app"
Étape 2 — Lancer l'app
Commande
Démarrer le serveur local
Tape cette commande et appuie sur Enter. Attends que tu voies "Ready" dans le terminal.
npm run dev
Étape 3 — Voir l'app
Dans ton navigateur
Ouvre cette adresse
L'app tourne localement sur ton ordi — personne d'autre ne peut la voir.
http://localhost:3000
Arrêter l'app
Dans le terminal
Ctrl + C
Ça arrête le serveur. L'app ne sera plus accessible sur localhost:3000.
💡 Astuce : Garde toujours le terminal ouvert pendant que tu travailles. Si tu fermes le terminal, l'app s'arrête automatiquement.
